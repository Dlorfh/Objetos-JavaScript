Objetos JavaScript

Um objeto js é uma coleção de propriedades, e uma propriedade é uma associação emtre um nome (ou chave) e um valor. Ou o valor de uma propriedade pode ser uma função, que é então considerada u m método do objeto

Os objetos js são udados para armazenar dados e fincuionalidades relacionados. Eles podem ser criados usando a sintaxe de objeto literal, a função construtora ou a classe.

Exemplo de objeto literal: 

        
            const pessoa = {
                nome: "Lorenzo",
                idade: 17,
                saudacao: function() {
                    console.log("Olá, meu nome é" + this.nome);
                }
            }
